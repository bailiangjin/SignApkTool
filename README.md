# SignApkTool
##Android sign apk tool
##Android  apk 签名工具
1、带图形化界面的 apk签名工具  
2、任意选择自由签名，再不被坑爹的写死签名文件的 签名工具所困扰  
3、只需配置JDK 环境变量 即可使用，支持jdk1.6 及更改版本Java环境  
4、mac windows 通杀  
5、支持对已签名apk的重签名，签名之前会自动去除已有签名无需手动操作
